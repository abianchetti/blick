Blick
=====

Open Source Gaze Tracker

    by Arturo Bianchetti and Silvia Comastri
    
    Developed using OpenCV and cvblob (Cristobal Carnero Li\~nan)
    Capabilities:   - gaze tracking.
                    - pupil diameter measurement.
                    - video rendering.
